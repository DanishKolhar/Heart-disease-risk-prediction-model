# 🫀 Heart Disease Prediction System

A Machine Learning-based web application that predicts the likelihood of heart disease using patient health parameters. The project uses a K-Nearest Neighbors (KNN) classification model and provides an interactive user interface built with Streamlit.

## 📌 Overview

Heart disease remains one of the leading causes of mortality worldwide. Early risk assessment can help individuals seek timely medical attention and adopt healthier lifestyles.

This project leverages Machine Learning techniques to analyze patient health data and predict whether a person is at risk of heart disease.

## 🚀 Features

* Interactive Streamlit-based user interface
* Real-time heart disease risk prediction
* Machine Learning model trained using K-Nearest Neighbors (KNN)
* Data preprocessing and feature scaling
* One-hot encoding for categorical features
* User-friendly medical input forms
* Instant prediction results

## 📊 Input Parameters

The model uses the following health indicators:

* Age
* Sex
* Chest Pain Type
* Resting Blood Pressure
* Cholesterol Level
* Fasting Blood Sugar
* Resting ECG Results
* Maximum Heart Rate Achieved
* Exercise-Induced Angina
* ST Depression (Oldpeak)
* ST Slope
